# ЕСИА Информация о подписи

При отправке запроса к ЕСИА в endpoin "../ac" параметр <client_secret> содержит электронную подпись detached signature закодированую в формате base64 url safe. Данная программа позволяет проверить правельность формирования подписи и получить информацию из сертификата.

## Использование

В командной строке
```sh
esia_sign_info.exe <Длинное значение client_secret>
```
